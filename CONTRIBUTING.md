# Contributing to CheFu Academy SDK

We ❤️ contributions! Here’s how to get started.

## How to Contribute
1. Fork the repository.
2. Clone your fork:
   ```bash
   git clone https://github.com/CheFu-code/chefu-academy-sdk.git
   ```
3. Install dependencies:
   ```bash
   npm install
   ```
4. Create a new branch for your feature or bugfix:
   ```bash
   git checkout -b feature/my-feature
   ```
5. Make your changes with clear commit messages.
6. Run tests to make sure nothing is broken:
   ```bash
   npm test
   ```
7. Commit your changes:
   ```bash
   git add .
   git commit -m "feat: add new feature"
   ```
8. Push to your fork and create a pull request.

## Code Guidelines
- Follow TypeScript conventions.
- Keep code modular and readable.
- Include JSDoc/type annotations for all public functions.
- Lint code using `npm run lint` before committing.

## Reporting Issues
If you find a bug or have a feature request, please open an issue on GitHub:
[CheFu Academy SDK Issues](https://github.com/CheFu-code/chefu-academy-sdk/issues)
